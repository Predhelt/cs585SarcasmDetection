How to run the program twitch-chatlog:
run line in cmd: 
-s = start time, -l = length in seconds
twitch-chatlog -s 00:15:00 -l 7200 <vod_id> > <filename>

twitch-chatlog -s 00:15:00 -l 7200 <vod_id> > "C:\Users\Ethan Johnson\Documents\GitHub\cs585SarcasmDetection\vod<streamerName>_output.txt"

ex:
twitch-chatlog -s 00:15:00 -l 10800 333145457 > "C:\Users\Ethan Johnson\Documents\GitHub\cs585SarcasmDetection\vodSips_3hrOutput.txt"
